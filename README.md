# Python Task Manager

A simple command-line task manager developed in Python as a personal learning project.

## Features

- Add tasks
- View all tasks
- Delete tasks
- Save tasks using a JSON file

## Technologies

- Python 3
- JSON

## How to Run

1. Clone the repository:
   git clone https://github.com/pmaganga05/python-task-manager.git

2. Navigate to the project folder:
   cd python-task-manager

3. Run the program:
   python main.py

## Purpose

This project was created to practice:
- Python fundamentals
- Functions
- File handling
- JSON data storage
- Git and GitHub
