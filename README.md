\# Python Task Manager



A simple command-line task manager built with Python. This project allows users to add, view, and delete tasks while saving data locally in a JSON file.



\## Features



\* Add new tasks

\* View all tasks

\* Delete existing tasks

\* Save tasks automatically using a JSON file

\* Simple and beginner-friendly command-line interface



\## Technologies Used



\* Python 3

\* JSON for data storage



\## Project Structure



```text

python-task-manager/

│

├── main.py

├── tasks.json

└── README.md

```



\## How to Run



1\. Clone the repository:



```bash

git clone https://github.com/pmaganga05/python-task-manager.git

```



2\. Navigate to the project folder:



```bash

cd python-task-manager

```



3\. Run the application:



```bash

python main.py

```
